Programs
===================

.. _cluster:

cluster
-------------------

Find and enumerate symmetry-equivalent clusters for cluster expansion [CE1]_ [CE2]_.

* Input file

  * :ref:`poscar.in`

* Output files

  * clusters.out, multiplicity.out, labels.out

* Command line option

  -d distance  truncation distance for searching symmetry-equivalent clusters
  -p file      POSCAR filename (default: poscar.in)

.. [CE1] J.M. Sanchez et. al., Physica A 128, 334-350 (1984).
.. [CE2] A. van de Walle and M. Asta, Modelling Simul. Mater. Sci. Eng. 10, 521-538 (2002).

getconf

Calculate properties of the system.


* Input file

	* :ref:`getconf.ini`, :ref:`poscar.spin`, clusters.in, multiplicity.in, labels.in,  :ref:`ecicar`

* Output file

	* Total energy, compositions and correlation functions.
