Welcome to wlce¶

WLCE (Wang-Landau sampling with Cluster Expansion) is an open source software for calculating alloy phase diagram and thermodynamic properties.

https://github.com/KazuhitoT/wlce/

Features¶

  • Canonical Metropolis sampling
  • Canonical Wang-Landau sampling
  • Semi-Grand-Canonical Wang-Landau sampling
