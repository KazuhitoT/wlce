3. Programs¶

3.1. cluster¶

Find and enumerate symmetry-equivalent clusters for cluster expansion [CE1] [CE2].

  • Input file
    • cluster.ini, poscar.in
  • Output files
    • cluster.out
[CE1]J.M. Sanchez et. al., Physica A 128, 334-350 (1984).
[CE2]
  1. van de Walle and M. Asta, Modelling Simul. Mater. Sci. Eng. 10, 521-538 (2002).

3.2. getconf¶

Calculate total energy or correlation functions of poscar.spin.

  • Input file

    • getconf.ini, poscar.spin, cluster.out, ecicar

  • Output file

    • Total energy, compositions and correlation functions.
